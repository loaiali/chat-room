* You have to run both the server and the client
* You have to install mongodb, create database named "ChatDB"

server:
1- if you don't have python3 install it
2- change dir to server/flask
3- run `pip install -r requirements.txt`
4- run `python __init__.py` to open the server 


client:
1- install node if you don't have it
2- change the dir to web-angular
3- run `npm install`
4- install angular-cli using `npm install -g @angular/cli`
5- run `ng serve -o` to open the web in the browser


client-app
* This is a native (not expo) react-native project
* don't use expo
1- import the react-native project and run it (follow this link https://facebook.github.io/react-native/docs/getting-started)
